1. Productivity & Resource Management
   Duplicate Detector: Highlight tabs with the same URL and provide a "Close Duplicates" button to save memory.
   Manual Discard (Tab Hibernation): Add a "Sleep" icon to manually trigger chrome.tabs.discard(id). This frees up RAM without closing the tab.
   Batch Actions: Add checkboxes or a "Close all from this domain" option when hovering over a tab or domain group.
2. Advanced Organization
   Domain Grouping: Automatically group tabs by domain (e.g., all github.com tabs under one collapsible header).
   Tab Groups Integration: Support Chrome's native Tab Groups, allowing users to see group colors and names, or even create new groups from the sidepanel.
   Vertical Tree View: Show parent-child relationships (e.g., if you opened a link in a new tab from GitHub, indent that new tab under the GitHub tab).
3. Intelligence & Context (Volt Specific)
   AI Summarization: Add a "Summarize" button on hover that uses your backend/AI to give a 2-sentence TL;DR of the tab's content without switching to it.
   Auto-Categorization: Use AI to tag tabs as "Research," "Social," "Work," or "Shopping" and allow filtering by these categories.
   Search Content: Expand search to include the actual text content of open tabs, not just titles and URLs.
4. Enhanced UI/UX
   Tab Previews: Show a small thumbnail preview of the tab when hovering over its title in the sidepanel.
   Audio/Status Indicators: Add icons for tabs that are currently playing audio, recording (mic/camera), or are pinned.
   "Read Later" Queue: A dedicated section to drag-and-drop tabs you want to save for later, persisting them even after the browser closes (similar to a temporary bookmark).
5. Navigation
   Sort Options: Allow sorting tabs by "Last Accessed," "Alphabetical," or "Domain" instead of just the default browser order.
   Multi-Window Support: A toggle to view tabs across all open Chrome windows instead of just the current one.
